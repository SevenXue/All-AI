# ALL-AI：包含了关于AI相关的项目，以及实现的demo
### Gans
关于Gans的一些demo，有cyclegan，pix2pix，dcgan。
### Attention
利用keras实现《attention is all your need》中的模型。
### seq2seq
实现seq2seq模型，但是效果不好，用第三方模块可以调用。
### lstm
使用keras的lstm层实现一个简单的LSTM模型。
### road_generator
使用pix2pix模型，seq2seq模型实现建筑道路自动化生成。
